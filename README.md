**Technologies:** React.js, Spring Boot, JavaMail API, MySQL, Spring Security

- 🚀 Built email automation platform for job seekers
- ⚡ Reduced application time from 5 minutes to 10 seconds (95% efficiency gain)
- 📨 SMTP/IMAP integration with optimized inbox fetching (2-3 min → 5 sec)
- 🔐 Secure template management with Spring Security
- 📎 Auto-attachment feature with BLOB storage

- 💼 Reduced administrative workflow by 80%
